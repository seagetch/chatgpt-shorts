# 綴じ穴は三つある

## Review findings (2026-03-14)

### Findings

1. `confirmed breakage`
   - 旧版は `前作とほぼ同じ形の事件` になっていた。
   - 具体的には、`その晩の決まった時刻までに暴かれる不都合な真相`、`関係者が順に止めに行く構図`、`発見直後の説明を言葉遣いの差で崩す解決` まで、前作の骨格と近すぎた。

2. `structural weakness`
   - 旧版は導入から死体発見までが早すぎ、中盤が `二人の嘘を剥がす` だけで終わっていた。
   - 中間解が一度立ってから崩れる快楽が弱く、読者が逗留する余地が少ない。

3. `clue design weakness`
   - 決め手が `呼び方の差` に寄りすぎていた。
   - 紙片の三つ穴も出てはいるが、終盤ではほぼ言葉の違和感が勝ってしまい、タイトルに見合う物的強度が足りなかった。

4. `suspect-field weakness`
   - 他の容疑者にも秘密はあったが、事件の本体が `実務担当者の改ざん隠し` に集まりすぎ、真犯人の重心が比較的早く見えてしまった。

### Verdict

- 旧版は `works as a mystery but with fairness or logic weaknesses` ではなく、今回の比較基準では `works mainly as a structural echo of the previous case, not as a distinct new mystery`。

### Repair scope

- `case rewrite`
- 主役二人だけ維持し、事件の核・舞台・犯行動機・手掛かりの型・中盤の進行を全面的に組み替える。

## Repair policy

- 前作との重複を避けるため、以下を採用しない:
  - `九時` などの固定時刻までに公開される真相を止める構図
  - `言い回しの差` を最終決め手にする構図
  - 発見直後の説明そのものを主たる偽装の核にする構図
- 今回の中核は:
  - `展示前夜の真贋崩し`
  - `紙の物理痕`
  - `中間解としての古書商犯人説`
  - `本物/偽物の問題が、犯人の局所動機そのものへ反転する構図`

## Case card

- victim: 御厨聡一。私設装幀工房 `芦屋装幀所` の主任修復師。
- culprit: 真壁礼人。近代文学研究者。
- motive: 話題の `水城澪第三書簡` が、自分の捏造した偽作であると御厨に見抜かれ、学問的信用と地位を失う直前だった。
- why now: 御厨がその夜のうちに第三書簡を額から外して再検査し、翌朝の内覧会から外すと決めたため。
- crime time: 午後八時四十分ごろ
- crime place: 芦屋装幀所の修復室
- method: 背丸め槌で後頭部を殴打
- concealment: 榛原書房の旧在庫封筒を作業机に開いたまま残し、古書商の榛原周二が流通経路を隠すために殺したよう見せた。
- trick: 真相の核は `第三書簡の左端に隠れた三つの綴じ穴`。それが、この書簡が近代の三穴ルーズリーフから切り出された偽物だと示す。
- fatal flaw in the plan:
  - 御厨は殺される前、第三書簡だけを透過光台に載せ、裏打ちを半分まで剥がしていた。
  - その状態では、左端に均等な三つの半円痕が残っていることを、紙を見る者なら確認できる。
  - さらに、真壁自身の論文だけが、その書簡にしかない文言を学説の柱にしていた。
- detective's decisive point:
  - `榛原が怪しい理由` は流通経路だけだが、`三つ穴のある偽物が露見すると最も致命傷を負う者` は真壁しかいない。
  - よって古書商を向いた疑いは中間解にすぎず、事件の中心は `真贋` にあると反転する。

## 1. Truth table

- case truth:
  - 久条家が後援する文学展の前夜、私設工房 `芦屋装幀所` で、近代詩人水城澪の新出書簡三通が明日の内覧会に向けて仕上げられていた。
  - そのうち `第三書簡` は、真壁礼人が数年前に自説を補強するために作った偽作だった。近代風の紙に見せかけているが、元は現代の三穴ルーズリーフを切り出し、染みと折りを加えたものだった。
  - 古書商の榛原周二は来歴の怪しさを知っていたが、全面的な偽作だとは知らず、怪しい由来の品として流した。
  - 御厨は額装のやり直しの過程で、第三書簡の左端、裏打ち紙の下に三つの半円痕を見つけた。これは元の紙に三穴パンチがあった証拠であり、近代の書簡ではありえない。
  - 御厨はその夜、第三書簡を展示から外し、翌朝の内覧会までに館主と保険担当へ報告するつもりだった。
  - 真壁はそれを聞いて修復室へ戻り、説得に失敗して御厨を背丸め槌で殴り殺した。
  - 真壁は第三書簡を持ち去れなかったため、榛原書房の旧在庫封筒を机上に残し、流通経路を争点に見せかけた。
- culprit: 真壁礼人
- motive:
  - 偽作が露見すれば、著書・講演・大学での地位が崩れる。
  - 彼の代表論文は第三書簡の一節を根拠にしており、真贋が覆れば単なる誤認では済まない。
- local action triggers:
  - 夕食前、御厨が `第三だけ額から外す` と明言した。
  - 夕食後、御厨が第三書簡を透過光台に載せ、半分まで裏打ちを剥がしているのを見て、真壁は翌朝まで隠し切れないと悟った。
  - 榛原の在庫封筒が作業机にあり、古書商へ疑いを向ける材料として使えると判断した。
  - だが第三書簡は湿しを入れた直後で、急いで剥がすと破れかねず、廊下の足音も近づいたため持ち去りを断念した。
- method:
  - 修復室で御厨に撤回を求め、断られた直後、近くの棚にあった背丸め槌で後頭部を一撃した。
  - ほぼ即死で、大きな叫び声は出なかった。
- tools used:
  - 背丸め槌
  - 榛原書房の旧在庫封筒
  - 作業机の灯りと透過光台
- crime time: 午後八時四十分ごろ
- alibi construction:
  - 真壁は夕食後しばらく講演原稿を直していたと主張する。
  - 実際には数分離席して修復室へ行き、戻ってからは講演メモの赤入れを続けていた。
  - 封筒を机上に残すことで、榛原が来歴の穴を揉み消しに来た印象を強める。
- hidden facts:
  - 安西槙子は八時十分ごろ御厨を訪ね、第三書簡だけは明日の展示から外してほしいと頼み、断られた。
  - 榛原周二は八時二十五分ごろ修復室の前まで来て、自分だけが悪質業者に見える説明は避けてほしいと頼むつもりだったが、中で話し声がして引き返した。
  - 篠宮藍は夕方、別の書簡の洗浄で軽いにじみを出しており、その報告を先延ばしにしていた。
  - 殺害直後、水場へ向かう篠宮の足音が廊下に近づき、真壁は第三書簡の持ち去りを断念した。
  - 真壁は前年、第三書簡だけを一週間単独で預かり、翻刻論文を書いている。
  - 御厨は導入部で、透過光にかけると `紙の傷は嘘をつきません` と澄乃たちに語っている。
- accidental events:
  - 榛原の封筒が本当に作業机に置かれていたため、偽装がよく効いた。
  - 槙子の懇願が先に露見し、彼女も濃い容疑者になる。
- culprit's miscalculation:
  - 第三書簡を回収せず、そのまま透過光台に残した。
  - 偽装の中心を `来歴` に置いたが、事件の本体は `真贋` だった。
  - 真壁の学説だけが第三書簡の固有文言に依存していた。
- detective's final logic:
  - 机上の封筒は榛原犯人説を強めるが、封筒は `来歴` の問題しか指さない。
  - しかし御厨が本当に見つけたのは `第三書簡それ自体の偽物性` である。
  - 透過光で見ると、第三書簡の左端には均等な三つの半円痕がある。これは三穴ルーズリーフ由来の物理痕であり、近代書簡にありえない。
  - すると、御厨を止める必要が最も大きいのは、流通経路が怪しい榛原ではなく、その偽物の本文に自説を載せた真壁である。
  - 真壁は前年に第三書簡を単独管理しており、偽作を差し込める立場にもあった。
  - よって殺人の局所動機と偽作の利益が一致する真壁が犯人に確定する。

## 6. Verification table

### Physical

- movement and travel time work: はい。食堂、展示室、修復室は同じ建物内で近い。
- method and tools work: はい。背丸め槌は近くにあり、凶器として使える。
- cause of death or damage is consistent: はい。後頭部への強打で即死に近い。
- impossible-crime setup actually stands: 不要。密室トリックには依存しない。
- weather, sound, light, and sightlines fit: はい。屋内で、短い離席は見落とされうる。

### Behavioral / Reality-Sense

- ordinary observers would react plausibly: はい。展示前夜に、それぞれが自分の損失を避けるため個別に動くのは自然。
- no implausible confusion of voice / object / timing: はい。核は物理痕と利益構造であり、不自然な聞き違いや超人的記憶に依存しない。
- trick survives casual checking: はい。封筒だけ見れば榛原が怪しく、書簡自体を見直して初めて反転する。
- local social behavior feels normal enough: はい。内覧会前夜の体面維持と保身が自然に働く。

### Psychological

- culprit behavior is believable: はい。自説の核が偽作だと露見する直前の研究者が、短絡的に殺害へ走る動機は理解できる。
- each key action has a legible local motive: はい。槙子は展示差し替え、榛原は来歴弁明、藍は失敗隠し、真壁は偽作露見回避。
- witness memory style is believable: はい。澄乃は部屋の往来と作業状況を覚えている。
- detective is not implausibly slow: はい。麗華は中間解に乗ったうえで、書簡そのものを見直して反転する。
- no "say this earlier" frustration: はい。各人の隠し事は初手では言いにくく、追及後に開く形で自然。

### Informational

- needed facts appear before the solution: はい。透過光説明、真壁の学説依存、第三だけ外すこと、各人の動きは先に出る。
- not enough facts to solve too early: はい。透過光説明だけでは足りず、書簡再検査と motive 結合が必要。
- no decisive new fact in the solution: はい。解決は既出要素の再結合で、三つ穴そのものも発見時点から潜在している。
- clues function, not just decorate: はい。透過光説明、封筒、第三だけ作業中、論文依存がすべて働く。

### Structural

- middle does not stall: はい。三人の嘘を剥がし、中間解を立て、物証で反転する三段になっている。
- suspicion changes form as truth approaches: はい。槙子疑惑、榛原疑惑を経て、真贋問題へ焦点が移る。
- final explanation is not bloated: はい。封筒の偽装と三つ穴の意味に絞れる。
- near-solution pleasure exists before the answer: はい。`なぜ第三だけを外していたのか` が見えた瞬間に反転の手前が来る。
