# 帰る声

引っ越しの日の朝は、家の中の音だけが少しよそゆきになる。

箱のふたを閉める音。食器の重なる音。階段を下りてくる足音。凪は片方だけ靴下を履いたまま台所へ来て、開口一番、「申請証、見てない？」と言った。

「昨日、自分で青いファイルにはさんだでしょう」

母はフライパンから目を上げずに答えた。

「青いの二冊ある」
「角の折れてるほう」
「そこまで覚えてる？」
「母親をなんだと思ってるの」

凪は言われたとおり居間へ戻り、すぐに「あった」と言った。その返事のしかたが、小学校に上がる前から少しも変わらないので、母は火を弱めた。

最後の朝くらい何か特別なものをと思ったのに、結局、朝食はいつもと同じになった。卵、スープ、トースト。凪は席につくなり、「また卵」と言い、母は「最後の朝くらい文句を言わない」と返す。こういうやり取りまで、今日で終わる。

「ほんとに行くんだな」と母が言った。
「今さら？」
「今さらよ」
「訓練、三年受けたけど」
「実感が湧くまで三年かかったの」

凪は笑いながら、トーストをちぎった。十七歳になって、笑い方だけ先に大人に似てきた。

食べ終わるころ、凪は窓の外を見て言った。

「海って、どんな匂いなんだろ」

窓の向こうには、朝の光を受けた山並みがやわらかく霞んでいた。谷の下の街路樹まで、いつもどおり静かだった。

「写真で見るより、たぶん鈍い匂いよ」と母は言った。
「鈍い？」
「もっと言うと、古い雨みたいな」
「全然わかんない」
「じゃあ、行って嗅いできなさい」
「母さんのぶんまで？」
「そう。ちゃんと報告すること」

凪は、はいはい、と気のない返事をした。

昼まで荷造りは何度もやり直しになった。入れたはずの本がまた机に戻り、畳んだ服の山が崩れ、しまったはずのコードがどこからか出てくる。引っ越しというのは、持っていくものを決めるより、置いていくものの形が目につく作業だ。

戸棚の奥から、古い鍵が一本出てきた。銀色はすっかり曇り、歯の先が少し丸い。

「これ、まだ取ってたんだ」

凪がつまみ上げると、母は振り向いて、それを少しだけ懐かしそうに見た。

「捨てそびれただけよ」
「もらっていい？」
「何も開かないわよ、たぶん」
「開かなくてもいい。鍵は鍵だろ」

母は笑った。

「そういう理屈っぽさ、誰に似たの」
「探しものの場所を全部覚えてる人」

午後になると、部屋はかえって片づいて見えた。大きな荷物から先に運び出されていくからだ。棚の跡、机の日焼け、長く置かれていたものの四角い影。人が暮らした場所には、物より先に習慣が残る。

夕方、出発の時刻が近づいたころ、母は玄関に立ったまま言った。

「忘れものは」
「ない」
「申請証」
「ある」
「通信用タグ」
「ある」
「薬」
「ある」
「鍵」
「ある」

凪はポケットを軽く叩いた。

「よろしい」

母はそう言ったが、扉を開ける手は少しだけ遅かった。

いつものように、窓の外は夕焼けだった。遠い山の稜線が藍色に沈み、谷の街に一つずつ灯りがともっている。子どものころから何度見たかわからない景色だ。見飽きているはずなのに、今日に限って、妙にきれいだった。

凪は靴紐を結び終えてから、ふと顔を上げた。

「母さん」

「なに」

「景色、もう切っていいよ」

母は黙った。

「最後までこのままでよかったのに」と、少しして言った。
「最後だから、ほんとのほうを見ておきたい」

母は返事の代わりに、壁際のパネルへ手を伸ばした。

かすかな作動音がして、窓の向こうの夕焼けが、音もなくほどけた。

山並みが消えた。谷が消えた。街の灯りが薄い粒になって流れ、壁の端へ吸い込まれるように消えた。そうしてあらわれたのは夜ではなく、もっと大きなものだった。

世界は外ではなく、上へ続いていた。

家々がゆるやかな曲線を描いて空のほうへせり上がり、そのまま遠く頭上で逆さまに連なっていた。道路も、公園も、灯りのついた窓も、すべてが巨大な筒の内側に貼りついて、見上げた先でこちら側の世界とつながっている。筒の中心には黒が貫いていて、その向こうに、青い星が浮かんでいた。

雲を巻いた、ひどく静かな地球だった。

凪はそれを見て、息を止めたわけでも、驚いた声を上げたわけでもなかった。ただ、ずっと見たかったものをようやく見た人の顔で、しばらく黙っていた。

母が言った。

「見せるならもっと前でもよかったのにね」
「ううん」
「どうして」
「この家を出る朝までは、ふつうの家でいたかったから」

母は何も言わなかった。

代わりに、薄い硝子越しのような黒の中で、地球だけがゆっくり回っていた。

「最初の帰還群だものね」と母が言った。
「そんな大げさな言い方」
「大げさじゃないわよ。人がまたあそこへ住むの」
「住めるようにした人たちが先にいる」
「それでも、帰るのは帰るでしょう」

凪はポケットの中で鍵を握った。何も開かない鍵。けれど、今日持っていくにはちょうどいい気がした。

「母さんは降りないんだよね」
「私はここで十分よ」
「強がり」
「少しはね」

母は笑ったが、その笑い方は朝より少し薄かった。

「それに、私の体はもう、下の重力にうまく合わないの。あんたたちみたいにはいかない」
「じゃあ、なおさら報告するよ」
「海の匂い？」
「それも。空の高さも。風の重さも」

母は、うなずいた。

降下港までの通路は明るすぎるほど明るかった。壁際には見送りの人が並び、ガラスの向こうに細い艇がいくつも待機している。定時放送が流れ、第一帰還群の搭乗を告げたとき、凪はようやく、自分が家を出るだけでは済まない場所まで来ているのだと知った。

母は搭乗口の手前で止まった。

「向こうに着いたら、最初になんて言うか決めてる？」
「なにそれ」
「決めておいたほうがいいわよ。帰る先なんだから」
「ただいま、でいい？」
「ええ」
「地球に？」
「地球に」

凪は少し笑った。

「返事、あるかな」
「なくても、言いなさい」

呼び出し音が鳴った。

「行ってきます」
「いってらっしゃい」

そのあとで、母はつけ足すように言った。

「海の匂い、忘れないで」

艇は定刻より九分遅れて切り離された。窓の向こうで、さっきまで住んでいた世界がゆっくり遠ざかる。巨大な円筒の一部に、見慣れた居住区の灯りが並んでいた。そのどれか一つが自分の家だとわかっているのに、もう指では示せなかった。

降下の振動は思っていたより静かで、地球は近づくほどに、星ではなく地面の色になっていった。

やがて着地の衝撃が一度だけあり、艇内の気圧が変わった。開いたハッチの向こうから、空気が流れこんでくる。

誰かが小さく息をのんだ。

凪も吸い込んだ。

塩のきつさはなかった。もっと鈍く、湿っていて、土のにおいが少しまじっていた。たしかに、古い雨みたいだった。

凪は思わず笑って、耳元の通信へ言った。

「母さん、わかった。古い雨だ」

少し遅れて、母の声が届く。

『でしょう』

その一言で、地球は急に遠い星ではなくなった。

凪は降下艇の梯子を下り、初めて土の上に立った。靴底にかかる重さが、これまで知っていたどの重さとも違う。風が頬を押し、どこか遠くで水の音がした。

ポケットの中で、古い鍵が指先に触れた。

凪は青い空を見上げた。軌道上の家は、もう見えない。

それでもちゃんと聞こえるように、少しだけ大きな声で言った。

「ただいま」

返事は、すぐには来なかった。風が先に頬を撫で、遠くで波が砕けた。

それから遅れて、軌道の家から母の声が降ってきた。

『おかえり』
