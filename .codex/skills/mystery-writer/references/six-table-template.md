### 6. Verification table

#### Physical

- movement and travel time work:
- method and tools work:
- cause of death or damage is consistent:
- impossible-crime setup actually stands:
- weather, sound, light, and sightlines fit:

#### Behavioral / Reality-Sense

- ordinary observers would react plausibly:
- no implausible confusion of voice / object / timing:
- trick survives casual checking:
- local social behavior feels normal enough:

#### Psychological

- culprit behavior is believable:
- each key action has a legible local motive:
- witness memory style is believable:
- detective is not implausibly slow:
- no "say this earlier" frustration:

#### Informational

- needed facts appear before the solution:
- not enough facts to solve too early:
- no decisive new fact in the solution:
- clues function, not just decorate:

#### Structural

- middle does not stall:
- suspicion changes form as truth approaches:
- final explanation is not bloated:
- near-solution pleasure exists before the answer:
