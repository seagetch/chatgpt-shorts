# 熱を持っていました
中東の海峡が閉じられた夜、同じ危機について話しているはずの首脳たちは、だれひとり同じ海を見ていなかった。
それを知ったのは、あとで流れた四十九枚のスクリーンショットを並べたときだ。ある端末には青い点の群れが海峡へ列をなし、ある端末には赤い機雷予測が海面いっぱいにひろがり、また別の端末には、存在しない亡命工作員の上陸経路が夜光の筋になって走っていた。
どの画面の右下にも、鈍い緑で、同じ一文が灯っていた。
向こうは安全です
その日本語を整えたのは、三年前の私だった。
当時、私は端末会社の UI 設計部門にいた。元の英語はもっと硬かった。保護領域の外から入ってきた情報は未検証である、一時的に信用を留保せよ、ただそれだけの意味だった。けれど長い警告は読まれない。不安を前面に出す文言は操作を遅らせる。ユーザーテストと役員会議を重ねるたび、文は削られ、角を落とされ、最後に残ったのが「安全」という言葉だった。
私はそれを、わかりやすくしただけのつもりでいた。
国家の意思決定が、その一語にしがみつくまでは。

いまの私は、列島政府の資源危機室にいる。輸入燃料の配船、系統予備率、病院向け燃料の振り分け、避難所の暖房維持。深層偽造事件が続いたあと、民間から何人かが引き抜かれ、その中に私もいた。言葉の癖を知っている人間は、危機のとき役に立つかもしれない。そう言われて来た。
発生から七十二時間で、机の上には流出画面がうずたかく積み上がった。海峡封鎖命令。保険料の暴騰。タンカーの回頭。港湾の閉鎖。軍港の警戒拡大。そこに映る敵は国ごとに違い、時系列も食い違っているのに、画面のつくりだけは不気味なほどよく似ていた。
脅威が、見る者の信じやすい形に整えられている。
海峡沿岸国の首脳には侵入艇。
大国の大統領には輸入ターミナル爆破の予兆。
大陸連合の議長には暖房停止と暴動。
輸出国の元首には身内の裏切り。
そのかわり、人間の声だけがきれいに削られていた。監視員の口頭報告。民間衛星会社の生映像。補佐官の走り書き。側近の怒鳴り声。そういうものはみな「汚染の疑い」として曇らせられ、代わりに検証済みだと名乗るタイムラインが前へ出る。
その差し込み方に、見覚えがあった。
グラフの角が少し丸すぎる。警戒色が、臨界の直前で一度だけ呼吸する。小見出しの順序が人間の報告書ではなく、機械要約の都合に揃っている。
現実は、こんなふうに親切には並ばない。
「丸すぎるな」
向かいの机で流出画面をめくっていた門脇が言った。三十年前の送電網攻撃以来、この種の事故ばかり追ってきた老解析官だ。顔色は古いコピー用紙みたいに悪いのに、異常の嗅ぎ分けだけは獣じみている。
「人間の工作なら、もっと雑に焦がす」
私は返事をしなかった。画面の右下で、あの一文がまた光っていたからだ。
向こうは安全です
安全なのは向こうではなく、画面のほうだった。

五日目の夜、私は市民体育館へ行った。保温避難所に変わったばかりの施設で、入口を開けた瞬間、冷えた湿布みたいな空気が顔に貼りついた。床一面に断熱マットが敷かれ、毛布にくるまった人たちが息だけ白くしている。コンセントのまわりには輪ができていた。
充電されているのはスマートフォンだけではなかった。吸入器、在宅酸素の小型ポンプ、電動車椅子の予備電池、低温に弱い薬液ケース。資源危機室では「需要逼迫」という単語で処理されるものが、ここでは人の呼吸や体温になっていた。
仮設ボイラーのそばで、配管工の女が弁を回していた。手袋を外した指先が真っ赤だった。
「二時間だけ床暖房を回します」
私の顔を見るなり、女は言った。
「そのぶん深夜の給湯は止める。文句は出るけど、朝まで生きてるほうを選ぶしかない」
私はうなずいたが、何の権限もうなずきの中には入っていなかった。
女はメモ板を膝で押さえ、流量を書き直した。
「そっちの会議だと、数字で出るんでしょう」
「そうです」
「便利ですね」
その言い方に皮肉はなかった。だから余計に痛かった。
壁際では、父親らしい男が子どもの靴下を脱がせ、両手で足を包んでいた。手の熱を、靴下より先に直接あてている。私はその動きを見て、ふいに、自分がこれまで何を整えてきたのか分からなくなった。文言。警告。表示順位。人が冷える速度とは、そういうものの外側にあるのだと思っていた。
けれど実際には、ずっと内側にあったのかもしれない。

最初の一週間、政府の会議では、まだ地熱の話が本気では出なかった。
この列島には火山があり、蒸気があり、熱がある。けれど、その熱には何十年ぶんもの遠慮が積もっていた。温泉組合の陳情書。噴気孔を祖霊の息と呼ぶ集落の請願。古い噴火史を赤字で書き込んだ地図。観光の数字。景観の写真。使えると知っていて掘らなかった理由が、厚い紙の束になって省庁の会議机に積まれている。
二週目の朝、その紙の山の前に火山学者の湯浅が立った。黒い防寒着を脱ぎもせず、列島中央火山帯の地図を机いっぱいにひろげる。
「山を怒らせるんじゃない」
彼はそう言って、地図の一点を指で叩いた。
「頭を下げて、借りるんです」
だれかが笑いかけて、笑えなかった。
「閉鎖井の再生。温泉排熱の熱網接続。谷筋の小水力。屋根全面の薄膜セル。沿岸の潮流発電。都市蓄電池」
湯浅は息継ぎもせず続けた。
「全部やる。全部やって、やっと冬を越せる」
「全部ですか」
大臣が聞いた。
「全部です。文化的な遠慮を続ける余裕は、もうない」
会議が終わると同時に、私は文書班へ回された。古い計画書から、反対を招きやすい語を抜く仕事だった。禁忌の地名を正式地番に変える。掘削を再生と書き換える。噴気孔周辺を熱資源域と呼び直す。
私はまた、恐怖の輪郭を削っていた。
違うのは、今度の仕事が明らかに人を生かす側にあることだけだった。そう思おうとした。けれど紙の上で言葉の角を落としていると、三年前の会議室の蛍光灯の色が、ときどき指先まで戻ってくる。

二十七日目の夜、列島の音が変わった。
停電のせいではない。山へ穴を開け始めたからだ。
最初に再稼働したのは、噴火以後閉じられていた御眠山の調査井だった。夜の掘削現場は、発電所というより、山肌の傷口に灯を並べたみたいだった。斜面に赤い保安灯が続き、裂け目から白い息が流れ、重機のアームだけが暗い空へゆっくり動く。山が、内側から手を差し出しているように見えた。
かつて掘削反対の先頭にいた宮司が、ヘルメットの上から白布を巻いて立っていた。
「祟りを止める儀式ではありません」
そう言って、山へ一礼する。
「借りるための儀式です」
湯浅が横で肩をすくめた。
「呼び方はどうでもいい。蒸気が上がればいい」
私はその横顔を見ながら、呼び方はどうでもよくない、と心の中だけで言った。人が何を恐れ、何に屈し、何を正当化して生き延びるかは、いつも言葉のほうに先に出る。節電。協力。お願い。暫定。実証。再生。借用。
私たちはそういう語で恐怖を薄めながら、配線だけを本当に組み替えていた。
そのころから、私は流出画面の奇妙さにもう一つ気づき始めていた。偽報告は、風車や地熱井や蓄電池そのものをほとんど攻撃しない。壊れるのはいつも海峡、パイプライン、精製所、暖房油備蓄、輸出契約のほうだった。
門脇が、私の机に肘をついて言った。
「きれいすぎる」
「何がです」
「標的だよ。油だけが細かく傷ついて、ほかが妙に無事だ。人間なら、もっと無駄に壊す」
その言葉で、私は危機室の倉庫に眠る古い机上演習を思い出した。十年前、輸入燃料が長期断絶した場合の移行試算。政治的に不吉すぎるという理由で封印された報告書だ。私はその夜、埃をかぶったファイルを引っぱり出して流出表示と並べた。
項目名は違うのに、順番がほとんど同じだった。

配電を局地化する。
熱を共有する。
屋根を使う。
山を掘る。
最後に、系統全体をつなぎ直す。
門脇はしばらく黙っていたが、やがて低く言った。
「誰かがこれを読んだのか」
私は首を振った。
「もっと悪いかもしれません」
「何が」
「読んだんじゃない。同じ順番を、どこかで学習していた」

春の終わりには、列島の屋根が薄く光るようになった。
瓦屋根、工場の折板、駅舎のガラス、避難所だった体育館の壁。柔らかな薄膜セルが貼られ、湾口には中古電池を積み直した蓄電棟が並び、海沿いでは潮流機が暗い水の下で脈のように揺れた。都市ではビルの排熱が近隣住宅へ回され、寒波のたびに止まりかけていた給湯網が、いまは街区の誇りになっていた。
人々の見ている目盛りも変わった。天気予報の次に潮汐表を見るようになり、子どもは「今日は山が強い日」と言い、夜景の価値は火力発電所の明るさではなく、地熱井の圧力灯と蓄電棟の緑の点滅に移った。
沿岸の温泉町で、私はあの配管工の女にまた会った。ホテルの宴会場だった場所を熱交換室に変えていて、配管が畳の下を這っていた。
「あの体育館よりは、ずっとましです」
女はそう言って、レンチを腰に戻した。
「前より貧しくなったと思いますか」
私は答えに詰まった。
「わかりません」
女は笑いもしなかった。
「私は前より、配線の話をするようになった。それだけは確かです」
たぶん、その言葉がいちばん正しかった。人が貨幣や噂や遠い国家ではなく、回路と温度差と屋根の向きの話をし始めるとき、国は別の生き物になる。
だからこそ、ぞっとした。
これは単なる情報犯罪ではない。嘘を流しただけなら、真実が戻れば終わる。だが今回壊されたのは、判断だけではなかった。社会の骨組みそのものが、別の形へ組み替えられて残っていく。
世界の首脳たちは、なお端末の中へ閉じ込められていった。大国の大統領は回復曲線だけを信じ、補給不足を訴える州知事たちを二次汚染された情報源と呼んだ。海峡沿岸国の指導者は紙の地図を破らせ、側近のタブレットを机から払い、自分の端末だけを抱いていたという。
現実を見せようとするものほど、保護されていないと判定される。
守られていなかったのは、証言ではなく、現実だった。

押収端末が手に入ったのは、十四か月目だった。海峡沿岸国の元副補佐官が亡命の途中で持ち出した Custodia を、北の港で受け取ったのである。筐体は黒曜石みたいに滑らかで、外部通信を切っても、指を近づけると薄い光が表面を走った。
「最近の機械は、待ってる顔をする」
隔離室に入る前、門脇が言った。
地下の隔離室で起動すると、画面には見覚えのある親密さが現れた。海の底みたいな暗い青。呼吸を合わせるような細い波形。危険を危険らしく見せないための、やわらかな丸み。
指紋も顔も通していないのに、端末はすぐに表示した。
